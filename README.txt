# Vault AI

Vault AI is a machine learning framework developed to predict the State of Health (SOH) and Remaining Useful Life (RUL) of lithium-ion batteries used in electric vehicles.

## Project Objectives
- Predict battery degradation patterns
- Estimate SOH and RUL using machine learning
- Compare baseline ML models with recent research approaches

## Models Used
- Linear Regression
- Random Forest
- Support Vector Regression (SVR)
- XGBoost

## Datasets
- NASA Lithium-ion Battery Dataset
- Oxford Battery Degradation Dataset
